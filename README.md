# Wanderer Flow

Wanderer Flow is an open source no code flow builder for privacy friendly chatbot agents.

It is designed for building interactive conversation flows that can react to changing user input in real time without hidden logic or complex scripting.

## What Wanderer Flow is about

Wanderer Flow helps you build chatbot and agent experiences with:

* visual no code flow building
* transparent decision logic
* privacy by design
* reactive conversations that can handle corrections and context changes
* browser based execution
* easy sharing and embedding
* JSON based flows that are easy to understand and extend

Potential use cases include:

* therapy and intake flows
* product configurators
* legal guidance flows
* customer support
* surveys and research
* rapid prototyping
